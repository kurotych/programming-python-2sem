# 2. (П1) Виконання завдань з Python та sqlite3

## Мета практичного заняття

Закріпити навички роботи з SQLite за допомогою бібліотеки sqlite3.

## Завдання

Завдання виконувати лише за допомогою Python коду (без прямого втручання в БД)

### Завдання 1: Створення бази даних

Створіть базу даних для системи управління бібліотекою з такими таблицями:

- `books` (id, title, author, year, isbn)
- `readers` (id, name, email, phone)
- `loans` (id, book_id, reader_id, loan_date, return_date)

### Завдання 2: Заповнення даними

Додайте тестові дані до всіх таблиць (мінімум 5 записів у кожну).  
**Один з записів readers повинен мати ваше ім'я**

### Завдання 3: Запити

Напишіть запити та оформіть їх у вигляді функцій для:

1. Отримання всіх книг певного автора  
2. Пошук читачів, які мають незавершені позички (return_date is None) (Запис з вашим іменем повинен мати незавершену позичку)
3. Підрахунок кількості книг за роками видання  

### Завдання 4: Оновлення та видалення

Реалізуйте функції для:

- Оновлення інформації про книгу
- Видалення читача
- Повернення книги (оновлення дати повернення)
